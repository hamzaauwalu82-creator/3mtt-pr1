    # 3mtt-pr1
This project provides a foundational introduction to cloud computing through the practical setup of a Microsoft Azure environment.I have create an Azure Free Tier account and  access to a suite of cloud-native services including compute, storage, and networking. The goal is to familiarize aspiring DevOps and cloud engineers with the onboarding processes, resource management interfaces, and security frameworks used by major cloud providers.
                      
                      PROJECT GOALs
*   Establish a functional cloud environment using the Azure Free Tier.
*   Gain proficiency in navigating the Azure Portal and understanding its organizational structure.
*   Understand the fundamentals of cloud governance, including subscriptions and resource groups.
*   Learn basic Identity and Access Management (IAM) concepts, such as Role-Based Access Control (RBAC).
*   Develop an awareness of cloud cost management, regions, and the shared responsibility model for security.
*   Understand the differences between IaaS, PaaS, and SaaS service models within a real-world platform.

                          TASKs
  1.  Account Registration: Create a new Microsoft Azure account and complete the identity verification process, including phone and payment method validation.
  2.  Portal Exploration: Navigate the Azure Portal to familiarize yourself with the dashboard, service menus, and search functionality.
  3.  Governance Setup: Identify your primary Subscription and create a new Resource Group to act as a logical container for future services.
  4.  Identity Awareness: Use the Azure Active Directory (Azure AD) menu to explore user management and review the documentation on Role-Based Access Control (RBAC).
  5.  Region Selection: Research and identify the Azure regions and availability zones nearest to your geographic location to optimize latency.
  6.  Resource Deployment: Perform a test deployment of a basic resource, such as a Virtual Machine (IaaS) or a Storage Account, to understand the configuration and validation workflow.
  7.  Cost Monitoring Config: Located the Cost Management + Billing section and explore tools for setting up budgets or usage alerts to remain within the Free Tier limits.

                        SUBMISSION
